Control schemes:

When working on code with group members, it is best to divide the work so everyone can carry out their work efficiently. In other words, a good control scheme for management and efficiency would drastically clean up and expedite the coding process, allowing everyone to perform at their best.

Git

Git is a open source control system. With its branching system, people can work on different parts of their code while experimenting with ideas, with the ability to revert back to past code. Check out more information here.

Github

Github is a hosting service for Git repositories - or rather, an area to store code. Check out more information here. Once you've signed up for Github, create a repository and link your members (if any) to it.
